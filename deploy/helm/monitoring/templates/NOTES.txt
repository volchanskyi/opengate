opengate-monitoring ({{ .Release.Name }}) deployed to namespace {{ .Release.Namespace }}.

Out-of-band objects this chart references (create BEFORE / alongside install):

  1. Secret with Grafana + Telegram + Postgres credentials:
       kubectl -n {{ .Release.Namespace }} create secret generic {{ .Values.existingSecret }} \
         --from-literal=GF_SECURITY_ADMIN_PASSWORD=... \
         --from-literal=POSTGRES_PASSWORD=... \
         --from-literal=TELEGRAM_BOT_TOKEN=... \
         --from-literal=TELEGRAM_CHAT_ID=...

  2. Grafana dashboards + alerting ConfigMaps, built from the canonical files
     (NOT duplicated into the chart):
       kubectl -n {{ .Release.Namespace }} create configmap {{ .Values.grafana.dashboardsConfigMap }} \
         --from-file=deploy/grafana/provisioning/dashboards
       kubectl -n {{ .Release.Namespace }} create configmap grafana-alerting \
         --from-file=deploy/grafana/provisioning/alerting
