opengate ({{ .Release.Name }}) deployed to namespace {{ .Release.Namespace }}.

Prerequisites (install once per cluster, outside this chart):
  1. ingress-nginx — snippet annotations stay DISABLED. The security headers are
     applied controller-side via the add-headers ConfigMap this chart renders
     (templates/custom-headers-configmap.yaml), so no per-ingress snippet (and no
     annotation-injection attack surface) is needed:
       helm install ingress-nginx ingress-nginx/ingress-nginx \
         --namespace ingress-nginx --create-namespace \
         --set controller.allowSnippetAnnotations=false \
         --set controller.config.add-headers={{ .Release.Namespace }}/{{ include "opengate.fullname" . }}-custom-headers
     On OKE, also annotate the controller Service for the Always-Free flexible LB
     (oci-load-balancer-shape=flexible, flex-min/max=10, subnet1=<lb-subnet>).
  2. cert-manager (CRDs + controller) for the ClusterIssuer:
       helm install cert-manager jetstack/cert-manager \
         --namespace cert-manager --create-namespace --set crds.enabled=true

Required Secret (NOT created by this chart — see secrets.example.yaml):
  kubectl create secret generic {{ .Values.server.existingSecret }} \
    --namespace {{ .Release.Namespace }} \
    --from-literal=JWT_SECRET=... \
    --from-literal=POSTGRES_PASSWORD=... \
    --from-literal=AMT_PASS=... \
    --from-literal=VAPID_CONTACT=mailto:ops@example.com{{ if .Values.postgres.backup.enabled }} \
    --from-literal=BACKUP_PAR_URL=https://objectstorage.<region>.oraclecloud.com/p/<token>/n/<ns>/b/opengate-pg-backups/o/{{ end }}
{{- if .Values.postgres.backup.enabled }}

Off-cluster backups (ADR-035) — create the bucket + write-only PAR + retention
lifecycle policy once, outside this chart, then put the PAR base URL (ending in
`/o/`) in the Secret key BACKUP_PAR_URL above:
  oci os bucket create --name opengate-pg-backups --compartment-id <ocid>
  oci os preauth-request create --bucket-name opengate-pg-backups \
    --name pg-backup-writer --access-type AnyObjectWrite \
    --time-expires <YYYY-MM-DD> --object-name ''      # write-only, no listing
  # Retention (replaces the old `find -mtime`): delete objects older than keepDays
  oci os object-lifecycle-policy put --bucket-name opengate-pg-backups \
    --items '[{"name":"expire","action":"DELETE","timeAmount":{{ .Values.postgres.backup.keepDays }},"timeUnit":"DAYS","isEnabled":true,"target":"objects"}]'
{{- end }}
